A simple vanilla JavaScript color flipper application built to practice DOM manipulation and event listeners.

## Features
- Click a button to cycle through a predefined array of colors  
- Switch to the **Hex** mode to generate completely random hex colors  
- Dynamically updates the background color and displays the current color value
